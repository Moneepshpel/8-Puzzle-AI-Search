# 8-Puzzle Solver using BFS and DFS

## Project Description
This project solves the classic 8-Puzzle problem using Artificial Intelligence search algorithms.

## Implemented Algorithms
- Breadth-First Search (BFS)
- Depth-First Search (DFS)

## How to Run
1. Make sure Python is installed
2. Run:
   python bfs.py
   python dfs.py
